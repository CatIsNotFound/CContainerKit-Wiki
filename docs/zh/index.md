# CContainerKit-Wiki

## 概述

CContainerKit 是基于 C11 标准的 C 语言容器集合。

## 特性

- 支持 C11 标准的 C 语言扩展库。
- 提供了各种容器的实现，如静态数组、动态数组、栈、队列等。
- 支持任意类型变量的存储和操作。
